# 人工演示步骤说明

## 推荐演示入口

从主菜单选择 `Chapter Run`。

## 推荐演示顺序

1. 主菜单
说明当前 build 状态、推荐入口和双关结构。

2. Stage 01
展示开场成长、火力升级、炸弹时机和第一关节奏。

3. ChapterBriefing
说明生命、炸弹、火力的继承与章节交接。

4. Stage 02
展示风暴环境、敌群联动、Boss 压迫和终盘 `FINAL BREACH`。

5. ChapterEnding
展示 `SLICE VERDICT`、`FINAL PASS` 和章节结尾判断。

6. ChapterOutro
展示 `FINAL PACKAGE` 与 `NEXT STEP`，作为最终收束。

## 演示时建议强调

- 这不是单关 prototype，而是双关垂直切片候选
- `Stage 01` 和 `Stage 02` 的职责明确不同
- 章节外层包装已经成立
- 当前版本已适合评审或阶段性交付
