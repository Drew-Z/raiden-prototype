# 试玩者快速说明

## 这是什么

这是一个纵版街机射击 Demo，用来验证双关短局节奏、火力成长、炸弹救场、Boss 压迫和结算反馈是否成立。

它还不是最终商业版，美术和部分音频仍然是展示候选状态。

## 推荐试玩方式

1. 启动项目后，点击黄色的 `开始完整 Demo`。
2. 优先完整打一遍 `Stage 01 -> Stage 02`。
3. 失败也没关系，失败后可以按 `R` 重开。
4. 通关或失败后请看一下结果页，总结页也是本次试玩观察重点。

## 操作

- 移动：`WASD` / 方向键
- 炸弹：`Space` / `Shift` / `X`
- 射击：自动持续开火
- 暂停：`Esc` / `P`
- 重开：`R`

## 请重点感受

- 火力升级有没有明显变强。
- 炸弹是否像一个有价值的救场资源。
- Boss 是否有压迫感，但还能看懂。
- HUD 和提示文字是否挡住战斗。
- 音效是否舒服，还是会疲劳或刺耳。
- 结果页是否能让你理解自己这局打得怎样。
