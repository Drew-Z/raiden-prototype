# 公开 Demo 发布说明

## 版本

- 版本标识：`RC-0.4`
- 当前定位：公开 Demo 候选包
- 推荐入口：主菜单黄色按钮 `开始完整 Demo`
- 推荐路线：`Stage 01 -> ChapterBriefing -> Stage 02 -> ChapterEnding -> ChapterOutro`

## 玩法简介

这是一个 Godot 4.6.1 制作的纵版街机射击 Demo，重点验证短局节奏、火力成长、炸弹救场、敌群编排、Boss 压迫和双关章节收束。

当前 Demo 不是完整商业版。它的目标是让玩家在一条紧凑双关路线里判断核心玩法是否成立。

## 操作

- 移动：`WASD` / 方向键
- 炸弹：`Space` / `Shift` / `X`
- 射击：自动持续开火
- 暂停：`Esc` / `P`
- 重开：`R`

## 推荐试玩方式

1. 点击黄色按钮进入完整 Demo。
2. 尽量完整打一遍双关流程。
3. 失败后可以按 `R` 重开。
4. 通关或失败后请查看结果页和章节总结。
5. 试玩后填写反馈表。

## 当前亮点

- 双关完整闭环已经成立。
- 火力成长、炸弹资源和 Boss 终盘窗口已经有明确反馈。
- 第二关有风暴机关和 Boss 联动，区别于第一关。
- 主菜单、简报、结尾、总结和结果页已经形成完整外层流程。

## 当前限制

- 美术仍以占位和程序化表现为主。
- BGM 仍是程序化占位方向，不是最终音乐。
- 关卡长度是 Demo 体量，不代表商业版内容量。
- 当前主要面向 Windows 本地试玩验证。

## 反馈重点

- 是否能快速知道该点哪里开始。
- 是否理解炸弹和火力升级。
- Boss 是否有压迫感但仍然可读。
- 音效是否疲劳或刺耳。
- 结果页是否能总结本局表现。
