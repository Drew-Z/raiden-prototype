# 最终演示检查清单

## 推荐演示路径

优先从 `Chapter Run` 开始，而不是直接进单关。

推荐顺序：

1. 主菜单展示当前 build 状态
2. 进入 `Chapter Run`
3. 在 `Stage 01` 展示火力成长、炸弹资源和关卡交接
4. 在 `ChapterBriefing` 展示资源继承
5. 在 `Stage 02` 展示风暴机关、Boss 压迫和 `FINAL BREACH`
6. 让 `ChapterEnding` 与 `ChapterOutro` 完整播放

## 演示重点

- `Stage 01`：火力成长、炸弹时机、开场节奏
- `Stage 02`：环境互动、敌群压迫、Boss 终盘三段节拍
- `ChapterBriefing / ChapterEnding / ChapterOutro`：章节交接、封版判断和最终总结

## 演示时不要跳过的内容

- `ChapterBriefing` 的 carry-state
- `Stage 02` 的 storm hazard 与 Boss 联动
- `ChapterEnding` 的 `SLICE VERDICT` 与 `FINAL PASS`
- `ChapterOutro` 的 `FINAL PACKAGE` 与 `NEXT STEP`

## 当前结论

当前 build 已经适合用作双关垂直切片候选的演示版本。后续如果继续推进，优先级应是包装和封版，而不是继续扩系统。
