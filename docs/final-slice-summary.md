# 双关垂直切片候选封版总结

## 当前结论

当前版本已经不是单关玩法原型，而是一个可完整演示的双关垂直切片候选。

- `Stage 01` 负责建立成长、炸弹节拍和章节交接
- `Stage 02` 负责展示更强的环境互动、Boss 压迫和终盘收束
- `Chapter Run` 已经形成 `Briefing -> Combat -> Ending -> Debrief` 的完整链路

## 当前完整体验

玩家可以从主菜单直接进入单关或双关章节模式。

- `Stage 01` 提供稳定的火力成长和敌群节奏
- 第一关结束后进入独立 `ChapterBriefing`
- `Stage 02` 会把风暴机关、敌群组合和 Boss 段落收成更强的高潮
- 双关结束后依次进入 `ChapterEnding` 与 `ChapterOutro`

## 已成立的亮点

- 火力成长、炸弹资源和短局节拍已经成立
- 双关章节的开始、交接、结束都由独立 scene 承接
- 第二关 Boss 已经具有 `LAST SAFE WINDOW -> OVERDRIVE -> FINISH WINDOW -> FINAL BREACH` 的清楚节拍
- 当前 build 已经足够作为 review build 或展示 build 使用

## 仍然缺的部分

- 正式音效、音乐和更成熟的资源替换
- 更强的镜头驱动过场，而不只是 UI 包装
- 最终数值微调和更严格的演示稳定性检查

## 建议的封版动作

1. 冻结系统范围，不再继续加新机制
2. 只做最后一轮节拍、文案和 UI 包装微调
3. 用当前双关章节作为正式演示路径
4. 以这版为基础准备更正式的垂直切片评审
