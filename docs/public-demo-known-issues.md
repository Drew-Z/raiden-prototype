# 公开 Demo 已知问题

## 当前已知限制

- 美术资源仍然是展示级占位，不是最终商业资产。
- 背景音乐是程序化占位，主要用于验证节奏和氛围方向。
- 当前只有双关路线，内容量不代表最终商业版长度。
- 结果页和章节总结仍偏开发复盘风格，后续可继续压缩成更玩家化的表达。
- 当前主要验证 Windows 环境，尚未做完整手柄、不同分辨率和导出包矩阵测试。

## 技术提示

- Windows 上 Godot 偶尔会输出 `Failed to read the root certificate store`，目前不影响本地运行、验证和打包。
- 如果 Godot 进程残留导致启动异常，可以运行 `tools\clear_godot_processes.cmd` 后再重新启动。
- 打包脚本会生成随机后缀 zip，避免旧包被 Windows 权限或杀毒软件短暂锁住。

## 暂不处理项

- 多章节长流程。
- 完整商业美术替换。
- 商店、养成、复杂武器分支。
- 在线排行榜或平台成就。

## 优先收集反馈

- 首屏是否清楚。
- 战斗画面是否可读。
- 炸弹是否有存在感。
- Boss 是否太快或太慢。
- 声音是否舒服。
- 玩家是否愿意重开第二局。
