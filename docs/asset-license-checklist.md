# 素材授权检查清单

## 当前音频来源

- Kenney：当前记录为 CC0，适合 Demo 和商业验证使用。
- Shapeforms Audio Free Sound Effects：使用用户本地下载包，需保留包内许可文本，最终商用前再次确认条款。
- Sonniss GDC Bundle：使用用户本地下载包，需保留官方包许可说明，最终商用前再次确认条款。

## 当前程序化资源

- 玩家、敌人、Boss、背景、子弹、爆炸、HUD、菜单背景主要由项目脚本绘制。
- 程序化 BGM 由项目脚本生成，目前作为占位音乐方向使用。

## 公开 Demo 前必须保留

- `docs/audio-sources.md`
- 本文件
- 外部素材原始许可文本的本地备份

## 商业版前必须完成

- 逐项确认所有音频素材的商用许可。
- 确认字体、Logo、图标和截图中没有未授权内容。
- 如替换正式美术，记录作者、来源、授权和使用范围。
- 如果使用素材包，保留下载页面、许可文本和版本日期。

## 当前风险

- Shapeforms 和 Sonniss 的具体许可需要以用户本地下载包内文本为准，不能只依赖记忆。
- 当前 Demo 可以继续用于小范围试玩验证；如果要公开大范围分发或商业化，需要再做一次完整授权复核。
